# Price Tags Generator

Система для создания и генерации ценников с помощью SVG шаблонов.

## Возможности

- **Конструктор шаблонов** - создание SVG шаблонов ценников с визуальным редактором
- **Управление товарами** - добавление и редактирование товаров
- **Генерация ценников** - автоматическое создание ценников на основе шаблонов
- **Интерактивный превью** - масштабирование и навигация по ценникам
- **Экспорт SVG** - сохранение готовых ценников в формате SVG

## Технологии

### Backend
- **Go** - основной язык сервера
- **Gin** - веб-фреймворк
- **File storage** - локальное хранение файлов

### Frontend  
- **Vue 3** - фронтенд фреймворк
- **TypeScript** - типизация
- **PrimeVue** - UI компоненты
- **Pinia** - управление состоянием
- **Vite** - сборщик

## Запуск проекта

### Backend (Go)
```bash
cd backend
go mod tidy
go run main.go
```

### Frontend (Vue)
```bash
cd front
npm install
npm run dev
```

## Использование

1. **Создание шаблона** - используйте конструктор для создания SVG шаблона ценника
2. **Добавление товаров** - создайте товары с названием и ценой
3. **Генерация ценника** - выберите шаблон и товар для создания ценника
4. **Экспорт** - сохраните готовый ценник в формате SVG
